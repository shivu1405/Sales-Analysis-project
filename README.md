# Superstore Sales Analysis

A SQL-based sales analysis project built using SQLite and Python.

## What this project covers
- Regional sales vs profit comparison across 4 US regions
- Root cause analysis of Central region's low profitability
- Product category profit margin analysis

## Key Findings
- Central region discounts at 24% — more than double the West (10.9%)
- Central makes only $17 profit per order vs West's $34
- Furniture has a 2.5% profit margin despite $742K in sales
- Technology is the most efficient category at 17.4% margin

Python · pandas · SQLite · matplotlib · Google Colab
